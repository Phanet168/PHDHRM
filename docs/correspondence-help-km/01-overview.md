п»ҝ# бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһ“бҹғбһҳбҹүбһјбһҢбһ»бһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбһ·бһҒбһ·бһҸбһҡбһҠбҹ’бһӢбһ”бһ¶бһӣ

## бҹЎ) бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’

бһҳбҹүбһјбһҢбһ»бһӣбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ **бһӣбһ·бһҒбһ·бһҸбһ…бһјбһӣ** бһ“бһ·бһ„ **бһӣбһ·бһҒбһ·бһҸбһ…бҹҒбһү** бһҠбҹ„бһҷбһҳбһ¶бһ“бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ–

- бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһӣбһ·бһҒбһ·бһҸбһұбҹ’бһҷбһҳбһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ
- бһ”бҹӮбһ„бһ…бҹӮбһҖбһ‘бҹ…бһўбһ„бҹ’бһӮбһ—бһ¶бһ–/бһўбҹ’бһ“бһҖбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- бһ”бҹ’бһҡбһҳбһјбһӣбһ…бҹҶбһҺбһ¶бһҡ бһ“бһ·бһ„бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһҸбһ¶бһҳбһҮбҹҶбһ бһ¶бһ“
- бһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҠбҹ„бһҷбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҠбһ№бһҖбһ“бһ¶бҹҶ
- бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбҹ’бһ“бһ¶бһҳбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ

## бҹў) бһҸбһҫбһўбҹ’бһ“бһҖбһҺбһ¶бһ”бҹ’бһҡбһҫбһўбҹ’бһңбһё?

### бһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһӣбһ·бһҒбһ·бһҸ (Clerk)
- бһ”бһ„бҹ’бһҖбһҫбһҸбһӣбһ·бһҒбһ·бһҸбһ…бһјбһӣ/бһ…бҹҒбһү
- бһ”бҹҶбһ–бҹҒбһүбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
- бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҜбһҖбһҹбһ¶бһҡ

### бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбһ·бһҒбһ·бһҸ (Admin Office)
- бһ…бһ¶бһҸбҹӢбһҸбһ¶бҹҶбһ„/бһ”бҹӮбһ„бһ…бҹӮбһҖбһ‘бҹ…бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’
- бһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”
- бһҡбҹҖбһ”бһ…бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбҹҒбһ…бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҠбһ№бһҖбһ“бһ¶бҹҶ

### бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’
- бһҳбһҫбһӣбһӣбһ·бһҒбһ·бһҸбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҒбҹ’бһӣбһҪбһ“
- бһ•бҹ’бһҸбһӣбҹӢбһ…бҹҶбһҺбһ¶бһҡ/бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ

### бһўбһ“бһ»бһ”бҹ’бһҡбһ’бһ¶бһ“бһҳбһ“бҹ’бһ‘бһёбһҡ
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бҹҶбһҺбһ¶бһҡ
- бһ•бҹ’бһҸбһӣбҹӢбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҠбһ№бһҖбһ“бһ¶бҹҶ

### бһ”бҹ’бһҡбһ’бһ¶бһ“бһҳбһ“бҹ’бһ‘бһёбһҡ
- бһўбһ“бһ»бһҳбҹҗбһҸ/бһ”бһҠбһ·бһҹбҹҒбһ’ (бһӣбһ·бһҒбһ·бһҸбһ…бһјбһӣ)
- бһҹбһҳбҹ’бһҡбҹҒбһ…бһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ

### Admin бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ (Super Admin)
- бһҖбҹҶбһҺбһҸбҹӢ Role/Permission
- бһҖбҹҶбһҺбһҸбҹӢ Org Role/Workflow Policy
- бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’

> бһ…бҹҶбһҺбһ¶бҹҶбҹ– бһ”бҹҠбһјбһҸбһ»бһ„бһҠбҹӮбһӣбһҳбһҫбһӣбһғбһҫбһүбһӣбһҫ UI бһҳбһ·бһ“бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ–бҹ’бһҡбҹ„бҹҮбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҹбһ·бһ‘бҹ’бһ’бһ· бһ“бһ·бһ„ Scope бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бҹ”

## бҹЈ) бһ‘бҹҶбһ–бҹҗбһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҢбһ»бһӣ

1. **бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӣбһ·бһҒбһ·бһҸбһ…бһјбһӣ**: бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһё бһ“бһ·бһ„бһ”бҹӮбһ„бһ…бҹӮбһҖбһӣбһ·бһҒбһ·бһҸбһ…бһјбһӣ
2. **бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӣбһ·бһҒбһ·бһҸбһ…бҹҒбһү**: бһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһ•бҹ’бһүбһҫбһӣбһ·бһҒбһ·бһҸбһ…бҹҒбһүбһ‘бҹ…бһўбҹ’бһ“бһҖбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’
3. **бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӣбһҳбҹ’бһўбһ·бһҸбһӣбһ·бһҒбһ·бһҸ**: бһҳбһҫбһӣбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бҹҒбһү + attachment + action history
4. **бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ…бҹҶбһҺбһ¶бһҡ/бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ**: бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҷбҹ„бһ”бһӣбҹӢбһҡбһ”бһҹбҹӢбһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһ“бһёбһҳбһҪбһҷбҹ—
5. **Print layout**: бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҡбһҠбҹ’бһӢбһ”бһ¶бһӣ

## бҹӨ) бһ–бһ¶бһҖбҹ’бһҷбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ— (UI Vocabulary)

- **To (individuals/org units)** = бһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһҹбҹҶбһҒбһ¶бһ“бҹӢ бһҸбҹ’бһҡбһјбһңбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ„бһ¶бһҡ
- **CC (individuals/org units)** = бһўбҹ’бһ“бһҖбһҮбһјбһ“бһ…бһҳбҹ’бһӣбһ„ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„
- **бһ…бһ¶бһҸбҹӢбһҸбһ¶бҹҶбһ„/бһ”бҹӮбһ„бһ…бҹӮбһҖ** = бһ”бҹ’бһҡбһӮбһӣбҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бҹ…бһўбҹ’бһ“бһҖ/бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ
- **бһ…бҹҶбһҺбһ¶бһҡ/бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ** = бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ¬бһҹбҹҶбһҺбһҫбһҹбһҳбҹ’бһҡбҹҒбһ…
- **Action history** = бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһўбҹ’бһңбһё бһ–бҹҒбһӣбһҺбһ¶ бһ“бҹ…бһҮбҹҶбһ бһ¶бһ“бһҺбһ¶
- **Completed** = бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ

---

## бҹҘ) бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһӣбһ·бһҒбһ·бһҸбһҠбҹӮбһӣбһҮбһҪбһ”бһүбһ№бһҖбһүбһ¶бһ”бҹӢ

- **Draft**: бһ‘бһҫбһ”бһ”бһ„бҹ’бһҖбһҫбһҸ бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ•бҹ’бһүбһҫ
- **Pending/In progress**: бһҖбҹҶбһ–бһ»бһ„бһҶбҹ’бһӣбһ„бһҮбҹҶбһ бһ¶бһ“ workflow
- **Sent/Distributed**: бһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһ‘бҹ…бһўбҹ’бһ“бһҖбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’
- **Approved**: бһ”бһ¶бһ“бһўбһ“бһ»бһҳбҹҗбһҸ (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбһ·бһҒбһ·бһҸбһ…бһјбһӣ)
- **Rejected**: бһ”бһҠбһ·бһҹбҹҒбһ’ бһҠбҹ„бһҷбһҳбһ¶бһ“бһҳбһјбһӣбһ бҹҒбһҸбһ»
- **Completed/Closed**: бһ”бһүбҹ’бһ…бһ”бҹӢбһӣбҹҶбһ бһјбһҡ

---

## бҹҰ) Quick Start (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҗбҹ’бһҳбһё)

### A. бһ”бҹ’бһҡбһҫбһӣбһ·бһҒбһ·бһҸбһ…бһјбһӣ
1. бһ…бһјбһӣ `бһӣбһ·бһҒбһ·бһҸбһ…бһјбһӣ` -> `бһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбһ·бһҒбһ·бһҸбһ…бһјбһӣ`
2. бһ”бҹҶбһ–бҹҒбһүбһңбһ¶бһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ + бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҜбһҖбһҹбһ¶бһҡ
3. бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ
4. бһ…бһ¶бһҸбҹӢбһҸбһ¶бҹҶбһ„бһ‘бҹ…бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’
5. бһ”бҹ’бһҡбһҳбһјбһӣбһ…бҹҶбһҺбһ¶бһҡ -> бһўбһ“бһ»бһ”бҹ’бһҡбһ’бһ¶бһ“бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ -> бһ”бҹ’бһҡбһ’бһ¶бһ“бһҹбһҳбҹ’бһҡбҹҒбһ…
6. бһ”бһүбҹ’бһ…бһ”бҹӢ/бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–

### B. бһ”бҹ’бһҡбһҫбһӣбһ·бһҒбһ·бһҸбһ…бҹҒбһү
1. бһ…бһјбһӣ `бһӣбһ·бһҒбһ·бһҸбһ…бҹҒбһү` -> `бһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбһ·бһҒбһ·бһҸбһ…бҹҒбһү`
2. бһ”бҹҶбһ–бҹҒбһүбһӣбҹҒбһҒбһӣбһ·бһҒбһ·бһҸ/бһҗбҹ’бһ„бҹғбһӣбһ·бһҒбһ·бһҸ/бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘/бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡ
3. бһҮбҹ’бһҡбһҫбһҹ To/CC (бһўбһ„бҹ’бһӮбһ—бһ¶бһ– бһ“бһ·бһ„бһ”бһ»бһӮбҹ’бһӮбһӣ)
4. бһ•бҹ’бһүбһҫбһ…бҹҒбһү
5. бһҸбһ¶бһҳбһҠбһ¶бһ“бһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“/бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ‘бһ‘бһҪбһӣ
6. Completed бһ“бҹ…бһ–бҹҒбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ„бһӣбһҠбҹ…

## бҹ§) бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһ¶бһҡбһ„бһ¶бһҡбһҳбһ·бһ“бһҮбһ¶бһ“бҹӢбһӮбҹ’бһ“бһ¶

- бһҖбһ»бҹҶбһ”бһүбҹ’бһ…бһјбһӣбһўбҹ’бһ“бһҖбһҠбһҠбҹӮбһӣбһҮбһ¶бһ‘бһ¶бҹҶбһ„ To бһ“бһ·бһ„ CC бһ”бҹ’бһҡбһҹбһ·бһ“бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ
- бһҳбһ»бһ“бһўбһ“бһ»бһҳбҹҗбһҸ бһҸбҹ’бһҡбһјбһңбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ attachment бһ“бһ·бһ„бһ…бҹҶбһҺбһ¶бһҡ бһұбҹ’бһҷбһӮбҹ’бһҡбһ”бҹӢ
- бһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹӮбһӣбһҮбһ¶бһӣбҹҶбһ бһјбһҡбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһҹбһјбһҳбһ’бҹ’бһңбһҫбһ“бҹ…бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҸбҹӮбһҳбһҪбһҷ (бһҳбһ·бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ action бһҹбҹ’бһ‘бһҪбһ“)
- бһ”бҹ’бһҡбһҫ Action history бһҮбһ¶бһ—бһҹбҹ’бһҸбһ»бһҸбһ¶бһ„бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ

- [ ] Role/Permission бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

- [ ] Workflow Policy Matrix бһҳбһ¶бһ“бһҮбҹҶбһ бһ¶бһ“бһ–бһ·бһҸ
