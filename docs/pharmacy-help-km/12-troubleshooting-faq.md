# បញ្ហាញឹកញាប់ និង FAQ

## បញ្ហាញឹកញាប់

### មិនឃើញមុខឱសថក្នុងបញ្ជី

មូលហេតុអាចជា:

- មិនទាន់បានបញ្ចូលក្នុង Medicines
- ឱសថត្រូវបានកែប្រែ ឬលុប

វិធីដោះស្រាយ:

- ពិនិត្យ Medicines ដោយស្វែងរកតាម Code ឬ Name
- បង្កើតឱសថថ្មី ប្រសិនបើមិនទាន់មាន

### ស្តុកមិនត្រឹមត្រូវ

មូលហេតុអាចជា:

- Distribution មិនទាន់បានកត់ត្រា
- Dispensing មិនទាន់បានកត់ត្រា
- Adjustment មិនទាន់បានបញ្ចូល
- Quantity ត្រូវបានបញ្ចូលខុស

វិធីដោះស្រាយ:

- ពិនិត្យ transaction ចុងក្រោយទាំងអស់
- ផ្ទៀងផ្ទាត់ batch និង quantity
- ប្រើ Adjustment នៅពេលចាំបាច់

### មិនអាច Auto-generate report បាន

មូលហេតុអាចជា:

- មិនទាន់ជ្រើស Facility
- មិនទាន់ជ្រើស Period start និង Period end
- មិនមាន transaction data ក្នុងកាលបរិច្ឆេទនោះ

វិធីដោះស្រាយ:

- ជ្រើស facility និង dates ឲ្យត្រឹមត្រូវ
- ពិនិត្យថាមាន distribution, dispensing ឬ adjustment ក្នុងរយៈពេលនោះឬទេ

### មិនអាច Save បាន

មូលហេតុអាចជា:

- ខ្វះ fields ចាំបាច់
- ទិន្នន័យមាន format មិនត្រឹមត្រូវ

វិធីដោះស្រាយ:

- ពិនិត្យ fields ដែលមានសញ្ញា *
- ពិនិត្យលេខ និងកាលបរិច្ឆេទឲ្យត្រឹមត្រូវ

## FAQ

### តើត្រូវបញ្ចូល Medicines មុន ឬធ្វើ Distribution មុន?

ត្រូវបញ្ចូល Medicines ជាមុនសិន។

### តើ Batch no ចាំបាច់ទេ?

មិនចាំបាច់គ្រប់ករណីទាំងអស់ទេ ប៉ុន្តែគួរបញ្ចូល ប្រសិនបើត្រូវតាមដាន batch និង expiry។

### តើ Closing stock គណនាយ៉ាងដូចម្តេច?

Closing stock = Opening + Received - Dispensed - Damaged - Expired + Adjust

### តើពេលណាគួរប្រើ adjustment type កែតម្រូវ?

ប្រើនៅពេលស្តុកជាក់ស្តែងខុសពីក្នុងប្រព័ន្ធ ហើយត្រូវកែសម្រួលចំនួនឲ្យត្រឹមត្រូវ។
