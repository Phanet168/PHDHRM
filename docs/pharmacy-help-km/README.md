## របៀបប្រើជាមួយ Help Center

- ប្រសិនបើចង់បង្ហាញ Help តាម menu អាចយកឯកសារទាំងនេះទៅ map ជា article នីមួយៗ
- ប្រសិនបើចង់បង្ហាញ Help តាម context អាចភ្ជាប់ឯកសារតាម route ឬ module name
