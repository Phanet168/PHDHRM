# Help សម្រាប់ Stock Adjustments

## គោលបំណង

Stock Adjustments ប្រើសម្រាប់កែសម្រួលស្តុកនៅពេលចំនួនក្នុងប្រព័ន្ធ មិនត្រូវនឹងស្តុកជាក់ស្តែង។

## ប្រភេទ Adjustment

- ខូចខាត: ដកចេញពីស្តុក
- ផុតកំណត់: ដកចេញពីស្តុក
- បាត់បង់: ដកចេញពីស្តុក
- កែតម្រូវ: បូកចូលស្តុក

## Fields សំខាន់ៗ

- Facility
- Medicine
- Type
- Qty
- Batch
- Expiry
- Date
- Reason

## ជំហានធ្វើ

1. ចូលទៅ Adjustments
2. ចុច New Stock Adjustment
3. ជ្រើស Facility
4. ជ្រើស medicine
5. ជ្រើស adjustment type
6. បញ្ចូល qty, batch, expiry និង date
7. បញ្ចូល reason
8. ចុច Add item ប្រសិនបើមានច្រើនមុខ
9. ចុច Save

## ឧទាហរណ៍ប្រើប្រាស់

- ឱសថខូចដោយសារកញ្ចប់បែក: ប្រើ ខូចខាត
- ឱសថផុតកំណត់: ប្រើ ផុតកំណត់
- ឱសថបាត់ក្នុងស្តុក: ប្រើ បាត់បង់
- រាប់ស្តុកឃើញលើស: ប្រើ កែតម្រូវ

## ការប្រុងប្រយ័ត្ន

- Type ខុស អាចធ្វើឲ្យស្តុកបូកឬដកខុស
- Reason គួរបញ្ចូលឲ្យច្បាស់សម្រាប់ audit
- Qty គួរតែផ្អែកលើការរាប់ស្តុកជាក់ស្តែង
