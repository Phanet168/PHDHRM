# ៣) បង្កើត/កែព័ត៌មានបុគ្គលិក

## 3.1 គោលបំណង

ផ្ទាំងនេះសម្រាប់បង្កើតបុគ្គលិកថ្មី និងកែព័ត៌មានបុគ្គលិកដែលមានស្រាប់ ដោយបែងចែកជាក្រុមព័ត៌មានច្បាស់ៗ។

## 3.2 រចនាសម្ព័ន្ធព័ត៌មាន (Tab/Section)

- ព័ត៌មានផ្ទាល់ខ្លួន
- ព័ត៌មានគ្រួសារ
- ប្រវត្តិសិក្សា/បណ្តុះបណ្តាល
- ព័ត៌មានធនាគារ
- ព័ត៌មានវេជ្ជសាស្ត្រ
- ព័ត៌មានអង្គភាព
- ឯកសារភ្ជាប់

## 3.3 ជំហានបង្កើតបុគ្គលិកថ្មី

1. បញ្ចូលអត្តសញ្ញាណ (ឈ្មោះ, ភេទ, DOB)
2. បញ្ចូលព័ត៌មានសញ្ជាតិ/ជនជាតិ
3. បញ្ចូលអាសយដ្ឋានកំណើត និងអាសយដ្ឋានបច្ចុប្បន្ន
4. បញ្ចូលព័ត៌មានទំនាក់ទំនង
5. បញ្ចូលព័ត៌មានការងារបឋម (អង្គភាព, តួនាទី)
6. បញ្ចូលឯកសារភ្ជាប់សំខាន់
7. ចុច Save ហើយពិនិត្យ View Profile

## 3.4 ចំណុចត្រូវប្រុងប្រយ័ត្ន

- អត្តលេខ 10 ខ្ទង់ មិនត្រូវស្ទួន
- ថ្ងៃផុតកំណត់ឯកសារ (ID/Passport/ប័ណ្ណបើកបរ) ត្រូវរក្សាទុកបានត្រឹមត្រូវ
- កុំដាក់ព័ត៌មានស្ទួនក្នុងក្រុមផ្សេងៗ
- ប្រសិនបើ field ជា dropdown សូមជ្រើសពី list ជាជាងវាយដោយដៃ

## 3.5 គោលការណ៍ Validation សំខាន់

- Required field (`*`) ត្រូវបំពេញមុន Save
- Date format ត្រូវតាមទ្រង់ទ្រាយប្រព័ន្ធ
- Email/Phone ប្រសិនបើបំពេញ ត្រូវមានទ្រង់ទ្រាយត្រឹមត្រូវ
- ឯកសារភ្ជាប់ត្រូវតាមប្រភេទ និងទំហំដែលបានអនុញ្ញាត

## 3.6 ករណីកែប្រែព័ត៌មាន

### កែប្រែទិន្នន័យធម្មតា

- កែបានភ្លាមៗ និង Save

### កែប្រែទិន្នន័យសំខាន់

- ដូចជា អង្គភាព, តួនាទី, ថ្នាក់, ស្ថានភាពការងារ
- ត្រូវប្រាកដថាបានកត់ត្រាក្នុងប្រវត្តិពាក់ព័ន្ធ
- ត្រូវមានលេខលិខិត/ថ្ងៃលិខិតយោង (បើ policy តម្រូវ)

## 3.7 Checklist មុនចុច Save

1. ពិនិត្យ Required fields
2. ពិនិត្យលេខអត្តលេខ 10 ខ្ទង់
3. ពិនិត្យ Date fields ទាំងអស់
4. ពិនិត្យអង្គភាព/តួនាទីឱ្យត្រូវ
5. ពិនិត្យឯកសារភ្ជាប់

## 3.8 បន្ទាប់ពី Save ត្រូវធ្វើអ្វី

1. ចូល View Profile ពិនិត្យទិន្នន័យបង្ហាញ
2. ពិនិត្យក្នុងបញ្ជីបុគ្គលិកថាមាន record ថ្មី
3. ពិនិត្យ notification/error log ប្រសិនបើមានកំហុស
