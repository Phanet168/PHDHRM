# ៥) ការឡើងតួនាទី

## 5.1 គោលបំណង

គ្រប់គ្រងការផ្លាស់ប្តូរតួនាទីជាផ្លូវការ និងរក្សាប្រវត្តិឱ្យពិនិត្យតាមក្រោយបាន។

## 5.2 ពេលណាត្រូវប្រើ

- ប្តូរតួនាទីដោយលិខិតសម្រេច
- តែងតាំងថ្មី/បន្ថែមភារកិច្ច
- កែសម្រួលតួនាទីដោយ policy អង្គភាព

## 5.3 ជំហានបង្កើតកំណត់ត្រាឡើងតួនាទី

1. ជ្រើសបុគ្គលិក
2. ជ្រើសតួនាទីថ្មី
3. បញ្ចូលថ្ងៃចាប់ផ្តើមតួនាទី
4. បញ្ចូលលេខលិខិត/ថ្ងៃលិខិត/ខ្លឹមសារលិខិត
5. Save

## 5.4 លទ្ធផលក្រោយ Save

- តួនាទីបច្ចុប្បន្នត្រូវបាន update
- ប្រវត្តិការងារបន្ថែម record ថ្មី
- អាចយកទៅរបាយការណ៍ និងលិខិតបោះពុម្ព

## 5.5 ច្បាប់ត្រូវប្រកាន់

- កុំប្ដូរតួនាទីបច្ចុប្បន្នដោយ edit ដោយផ្ទាល់បើមានម៉ូឌុលឯកទេស
- ត្រូវមានថ្ងៃមានប្រសិទ្ធភាពច្បាស់
- ត្រូវមានឯកសារគាំទ្រ

## 5.6 FAQ ខ្លី

- មិនឃើញតួនាទីថ្មី: ពិនិត្យ master data តួនាទី
- Save រួចតែនៅតែតួនាទីចាស់: ពិនិត្យថ្ងៃមានប្រសិទ្ធភាព និង conflict record
