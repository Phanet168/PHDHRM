# бһҹбҹҖбһңбһ—бҹ…бһҮбҹҶбһ“бһҪбһҷ: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ (HR Employee Management)

бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ HR/Admin бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһҠбһ№бһҖбһ“бһ¶бҹҶбһўбһ„бҹ’бһӮбһ—бһ¶бһ– бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбһ»бһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһұбҹ’бһҷбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҸбһ¶бһҳбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһҠбҹ’бһӢбһ”бһ¶бһӣ бһ“бһ·бһ„бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбҹҶбһ бһ»бһҹбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

## 1) бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбҹ„бһӣбһҠбҹ…

- бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“бһҳбһ“бһ»бһҹбҹ’бһҹ (HR/Admin)
- бһўбҹ’бһ“бһҖбһўбһ“бһ»бһҳбҹҗбһҸ (бһ”бҹ’бһҡбһ’бһ¶бһ“/бһўбһ“бһ»бһ”бҹ’бһҡбһ’бһ¶бһ“/бһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ•бҹ’бһ‘бҹҒбһҡбһҹбһ·бһ‘бҹ’бһ’)
- бһҳбһ“бҹ’бһҡбҹ’бһҸбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһҫбһӣбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҒбҹ’бһӣбһҪбһ“бһҜбһ„

## 2) бһӮбһ“бҹ’бһӣбһ№бҹҮбһ”бҹ’бһҡбһҫбһҜбһҖбһҹбһ¶бһҡ
