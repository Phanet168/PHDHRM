# លក្ខខណ្ឌគណនា ២ ឆ្នាំ (គិតត្រឹម ១ មេសា)

## គោលការណ៍គណនា

- កាលកំណត់គោល៖ **ថ្ងៃទី ១ ខែមេសា** នៃឆ្នាំជ្រើស
- ប្រព័ន្ធយក anchor date (ថ្ងៃដំឡើងចុងក្រោយ/ថ្ងៃចូលបម្រើការងារ) មកគណនា
- បើរយៈពេលគ្រប់ **២ ឆ្នាំឡើងទៅ** នឹងចូលបញ្ជីគ្រប់លក្ខខណ្ឌ

## ករណីត្រូវប្រុងប្រយ័ត្ន

- មន្ត្រីអសកម្ម ឬស្ថានភាពមិនរាប់អតីតភាព អាចមិនចូលបញ្ជី
- មន្ត្រីជិតនិវត្តន៍អាចចូលក្រុមលក្ខខណ្ឌពិសេស
- ករណីតាមសញ្ញាបត្រ ត្រូវធ្វើជាសំណើដាច់ដោយឡែក

## ពិនិត្យពេលឃើញ “ថ្នាក់បច្ចុប្បន្ន” និង “ថ្នាក់ត្រូវឡើង” ដូចគ្នា

- ពិនិត្យ Matrix លំដាប់ថ្នាក់ក្នុង DB (`pay_levels`)
- ពិនិត្យការចង mapping រវាងប្រភេទថ្នាក់ (ក/ខ/គ) មិនអោយឆ្លងគ្នា
- ប្រើការកែតម្រូវនៅ Tab បង្កើតសំណើ មុនអនុម័ត
