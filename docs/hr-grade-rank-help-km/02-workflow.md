# លំហូរការងារ

## ដំណាក់កាលស្នូល

1. បញ្ជីគ្រប់លក្ខខណ្ឌ
2. បង្កើតសំណើ
3. ការអនុម័ត
4. ប្រវត្តិដំឡើង

## សេចក្តីពន្យល់ខ្លី

### 1) បញ្ជីគ្រប់លក្ខខណ្ឌ

- ប្រព័ន្ធបង្ហាញបុគ្គលិកដែលឆ្លងលក្ខខណ្ឌតាមឆ្នាំ
- អាច Filter តាមអង្គភាព ប្រភេទបុគ្គលិក និងស្ថានភាព

### 2) បង្កើតសំណើ

- ជ្រើសបុគ្គលិកពីតារាង
- កំណត់ប្រភេទសំណើ ថ្ងៃមានប្រសិទ្ធភាព និងឯកសារយោង
- អាចបន្ថែម/ដកឈ្មោះមុន Submit

### 3) ការអនុម័ត

- អ្នកមានសិទ្ធិពិនិត្យសំណើ
- អាចអនុម័ត/បដិសេធ និងកត់មូលហេតុ
- អនុម័តរួច សំណើផ្លាស់ទៅប្រវត្តិ

### 4) ប្រវត្តិដំឡើង

- រក្សាទុកកំណត់ត្រាសកម្មភាពទាំងអស់
- ប្រើសម្រាប់ audit និងរបាយការណ៍ប្រចាំឆ្នាំ
