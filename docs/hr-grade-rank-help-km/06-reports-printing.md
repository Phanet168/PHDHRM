# របាយការណ៍ និងបោះពុម្ព

## របាយការណ៍

- ប្រើ Tab **របាយការណ៍** ដើម្បីមើលស្ថានភាពសរុប
- អាចចេញជា CSV/PDF តាមឆ្នាំដែលបានជ្រើស

## បោះពុម្ព

- ពិនិត្យទិន្នន័យអោយត្រឹមត្រូវមុន Print
- ប្រើ print layout ផ្លូវការដែលប្រព័ន្ធរៀបចំស្រាប់
- អក្សរខ្មែរ/Logo ប្រើតាម format រដ្ឋបាល

## ចំណាំ

- ប្រសិនបើអក្សរខ្មែរ display មិនត្រឹមត្រូវ សូមពិនិត្យ font setup នៅ server/client
- ប្រសិនបើលេខ/កាលបរិច្ឆេទមិនត្រឹមត្រូវ សូមពិនិត្យឆ្នាំ filter មុន export
