# Help Center សម្រាប់ការគ្រប់គ្រងថ្នាក់ និងឋានន្តរស័ក្តិ

- `01-overview.md`: ទិដ្ឋភាពទូទៅ និងគោលការណ៍

- `05-rules.md`: លក្ខខណ្ឌគណនា ២ ឆ្នាំ (គិតត្រឹម ១ មេសា)
