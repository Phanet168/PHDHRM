# ទិដ្ឋភាពទូទៅ

## គោលបំណងម៉ូឌុល

ម៉ូឌុលនេះប្រើសម្រាប់៖

- ត្រួតពិនិត្យមន្ត្រីដែលគ្រប់លក្ខខណ្ឌដំឡើងថ្នាក់/ឋានន្តរស័ក្តិ
- បង្កើតសំណើជាក្រុម ឬម្នាក់ៗ
- ដំណើរការអនុម័តតាមលំដាប់
- រក្សាទុកប្រវត្តិដំឡើង និងចេញរបាយការណ៍

## គោលការណ៍សំខាន់

- ការគណនាលក្ខខណ្ឌប្រចាំឆ្នាំ គិតត្រឹម **ថ្ងៃទី ១ ខែមេសា**
- ករណីដំឡើងប្រចាំឆ្នាំ ស្តង់ដារ​គឺ **២ ឆ្នាំឡើងទៅ** (ផ្អែកលើទិន្នន័យប្រវត្តិ)
- ប្រភេទសំណើដាច់ពីគ្នា៖
  - ដំឡើងប្រចាំឆ្នាំ
  - ដំឡើងតាមសញ្ញាបត្រ
  - ដំឡើងកិត្តិយសមុននិវត្តន៍

## អ្នកប្រើប្រាស់គោលដៅ

- HR/Admin អ្នករៀបចំបញ្ជីសំណើ
- អ្នកពិនិត្យ/អ្នកអនុម័តតាមតួនាទី
- អ្នកមើលរបាយការណ៍
