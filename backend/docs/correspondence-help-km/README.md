# Help Center សម្រាប់ម៉ូឌុល «គ្រប់គ្រងលិខិតរដ្ឋបាល»

ឯកសារជំនួយនេះរៀបចំសម្រាប់អ្នកប្រើប្រាស់មិនមែន IT និងអ្នកគ្រប់គ្រងប្រព័ន្ធ ដើម្បីអាចប្រើម៉ូឌុល **លិខិតចូល/លិខិតចេញ** បានត្រឹមត្រូវ តាមលំដាប់ការងាររដ្ឋបាល។

## របៀបអាន (ណែនាំ)

1. `01-overview.md` មើលទិដ្ឋភាពទូទៅ និងពាក្យសំខាន់
2. `02-incoming.md` អនុវត្តលំហូរ **លិខិតចូល**
3. `03-outgoing.md` អនុវត្តលំហូរ **លិខិតចេញ**
4. `04-workflow.md` យល់ពីជំហានសម្រេច និងសិទ្ធិគ្រប់តួនាទី
5. `05-feedback-notes.md` សរសេរចំណារ/មតិយោបល់អោយបានស្តង់ដារ
6. `06-troubleshooting-faq.md` ដោះស្រាយបញ្ហាញឹកញាប់
7. `07-role-permission-setup.md` កំណត់ Role/Permission/Org scope

## អ្វីដែលឯកសារនេះសំខាន់

- កាត់បន្ថយការច្រឡំពេលបែងចែកលិខិត
- ជួយអោយការសម្រេចចុងក្រោយមានស្នាមតាមដាន (audit trail)
- ជួយបណ្តុះបណ្តាលអ្នកប្រើថ្មីក្នុងពេលខ្លី
- ការងារអាចបន្តបានទោះប្តូរអ្នកកាន់ការងារ

## កំណត់សម្គាល់

- ប្រសិនបើ UI ប្តូរ សូមអាប់ដេតឯកសារ Help ជាមួយគ្នា
- សូមរក្សាពាក្យឯកភាពក្នុងប្រព័ន្ធ៖

  - ចាត់តាំង / បែងចែក / ជូនចម្លង
  - ចំណារ / មតិយោបល់ / អនុម័ត / បដិសេធ
