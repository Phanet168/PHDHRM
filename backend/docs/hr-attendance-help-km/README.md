# សៀវភៅជំនួយ: ការគ្រប់គ្រងវត្តមាន (Attendance SOP)

ឯកសារនេះសម្រាប់ HR/Admin និងអ្នកប្រើ Mobile ដើម្បីអនុវត្តវត្តមានឱ្យបានស្តង់ដារ និងកាត់បន្ថយកំហុស។

## អ្នកប្រើគោលដៅ

- HR/Admin
- ប្រធានអង្គភាព/អ្នកអនុម័ត
- បុគ្គលិកប្រើ Mobile សម្រាប់ Scan QR

## របៀបប្រើឯកសារ

1. អានតាមលំដាប់ `01 -> 05`
2. ប្រសិនបើជួបបញ្ហា សូមចូល `05-troubleshooting-faq.md`
3. ប្រសិនបើត្រូវកែសម្រួលវត្តមាន សូមចូល `04-exceptions-and-corrections.md`

## បញ្ជីឯកសារ

## សេចក្តីសង្ខេបស្តង់ដារ

- វត្តមានប្រើ `IN/OUT` ជា punch pair
- QR ត្រូវមានសុពលភាពខ្លី (Short expiry) និងពិនិត្យទីតាំង (GPS + Geofence)
- ករណី `PARTIAL` ឬ `UNPAIRED_PUNCH` ត្រូវតាមដានរាល់ថ្ងៃ
- រាល់ error scan ត្រូវមាន log សម្រាប់ audit
