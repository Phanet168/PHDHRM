# 01) SOP បុគ្គលិក (Mobile Employee Flow)

## គោលបំណង

ឱ្យបុគ្គលិកអាចចូលវត្តមានត្រឹមត្រូវ តាមអង្គភាព និងពេលវេលា។

## ជំហានប្រើប្រាស់

1. បើក App និង Login ដោយ `email/password`
2. ចូល `Attendance`
3. ចុច `Scan QR`
4. ដាក់ QR អង្គភាពក្នុងស៊ុមកាមេរ៉ា
5. អនុញ្ញាត GPS/Location
6. រង់ចាំប្រព័ន្ធបញ្ជូន (auto submit)
7. ពិនិត្យលទ្ធផល:
   - `Success`: បង្ហាញសារជោគជ័យ + Workplace + Distance + Allowed
   - `Error`: មើល message (ឧ. out_of_range, invalid_qr, no_gps)

## អ្វីដែលត្រូវពិនិត្យបន្ទាប់ពី Scan

- `Workplace` ត្រូវតែត្រូវនឹងអង្គភាពដែលបានស្កេន
- `Distance` តិចជាង `Allowed`
- កំណត់ត្រាថ្ងៃនេះបង្ហាញនៅ Attendance history

## កំហុសមិនគួរធ្វើ

- ស្កេន QR ចាស់ផុតសុពលភាព
- បិទ GPS ឬមិនអនុញ្ញាត Location
- ស្កេនក្រៅតំបន់អង្គភាព
- ស្កេនជាប់ៗគ្នាច្រើនដងក្នុងពេលខ្លី

## ប្រាប់អ្នកគ្រប់គ្រងពេលណា?

- បើលទ្ធផល Error ជាបន្តបន្ទាប់ (ច្រើនជាង 3 ដង)
- បើឃើញអង្គភាពមិនត្រឹមត្រូវក្នុង UI
- បើមានវត្តមានខ្វះទោះបីស្កេនជោគជ័យ
