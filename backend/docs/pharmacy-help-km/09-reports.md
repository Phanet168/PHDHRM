# Help សម្រាប់ Reports

## គោលបំណង

Reports ប្រើសម្រាប់បង្កើតរបាយការណ៍ឱសថតាម facility និងរយៈពេល។ អ្នកអាចបញ្ចូលរបាយការណ៍ដោយដៃ ឬប្រើ Auto-generate from data។

## Fields សំខាន់ៗ

- Facility
- Report to
- Report type
- Period label
- Period start
- Period end
- Note

ក្នុង Report items មាន:

- Opening
- Received
- Dispensed
- Damaged
- Expired
- Adjust
- Closing
- Note

## របៀបបង្កើតរបាយការណ៍ដោយដៃ

1. ចូលទៅ Reports
2. ចុច New report
3. ជ្រើស Facility និង Report type
4. បញ្ចូល period dates
5. បញ្ចូល report items ម្តងមួយមុខឱសថ
6. ពិនិត្យ closing stock
7. ចុច Save

## របៀប Auto-generate

1. ជ្រើស Facility
2. ជ្រើស Period start និង Period end
3. ចុច Auto-generate from data
4. រង់ចាំប្រព័ន្ធបំពេញ report items
5. ពិនិត្យទិន្នន័យម្តងទៀត
6. ចុច Save

## រូបមន្ត Closing stock

Closing stock = Opening + Received - Dispensed - Damaged - Expired + Adjust

## ការប្រុងប្រយ័ត្ន

- ត្រូវជ្រើស facility និង dates មុន auto-generate
- បើគ្មាន transaction ក្នុងរយៈពេលនោះ ប្រព័ន្ធអាចមិនមាន data
- សូមពិនិត្យ closing stock មុន submit ឬ print
