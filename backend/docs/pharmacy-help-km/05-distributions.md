# Help សម្រាប់ Distributions

## គោលបំណង

Distributions ប្រើសម្រាប់កត់ត្រាការផ្ទេរឱសថពី facility មួយទៅ facility មួយទៀត។ វាជាការកត់ត្រាចលនាស្តុកដែលសំខាន់សម្រាប់ traceability និង reporting។

## ព័ត៌មានត្រូវបញ្ចូល

- From facility
- To facility
- Distribution date
- Note
- Medicine items
- Quantity sent
- Batch no
- Expiry date
- Unit price

## ជំហានបង្កើត Distribution

1. ចូលទៅ Distributions
2. ចុច New distribution
3. ជ្រើស From facility និង To facility
4. បញ្ចូល Distribution date
5. បញ្ចូល Note ប្រសិនបើមាន
6. ជ្រើស medicine ក្នុងតារាង
7. បញ្ចូល quantity, batch, expiry និង unit price
8. ចុច Add item ប្រសិនបើមានច្រើនមុខ
9. ពិនិត្យទិន្នន័យទាំងអស់
10. ចុច Save

## ការប្រុងប្រយ័ត្ន

- មិនគួរជ្រើស From facility និង To facility ដូចគ្នា
- Quantity ត្រូវតែត្រឹមត្រូវ ព្រោះវាប៉ះពាល់ដល់ស្តុក
- Batch no និង expiry date គួរបញ្ចូលនៅពេលត្រូវតាមដាន lot

## កំហុសញឹកញាប់

- មិនបញ្ចូល quantity
- បញ្ចូល batch ខុសពីឯកសារដើម
- មិនពិនិត្យ expiry date មុន save
