# ទិដ្ឋភាពទូទៅនៃប្រព័ន្ធគ្រប់គ្រងឱសថ

## គោលបំណង

ប្រព័ន្ធគ្រប់គ្រងឱសថត្រូវបានប្រើសម្រាប់គ្រប់គ្រងបញ្ជីឱសថ ស្តុក ការចែកចាយ ការផ្តល់ឱសថ និងរបាយការណ៍។ គោលបំណងសំខាន់គឺធ្វើឲ្យទិន្នន័យឱសថមានភាពត្រឹមត្រូវ ងាយតាមដាន និងអាចប្រើសម្រាប់ការសម្រេចចិត្តបានល្អ។

## អ្នកប្រើប្រាស់គោលដៅ

- អ្នកគ្រប់គ្រងឱសថ
- អ្នកបំពេញស្តុក ឬអ្នកទទួលបន្ទុកឃ្លាំង
- អ្នកផ្តល់ឱសថដល់អ្នកជំងឺ
- អ្នកធ្វើរបាយការណ៍
- Admin ប្រព័ន្ធ

## មុខងារសំខាន់ៗ

- Dashboard: មើលទិន្នន័យសង្ខេបទូទៅ
- Medicines: គ្រប់គ្រង master data ឱសថ
- Categories: បែងចែកប្រភេទឱសថ
- Distributions: កត់ត្រាការផ្ទេរឱសថរវាង facility
- Stock: ពិនិត្យស្តុកបច្ចុប្បន្ន
- Adjustments: កែសម្រួលស្តុក
- Dispensing: ផ្តល់ឱសថដល់អ្នកជំងឺ
- Reports: បង្កើតរបាយការណ៍លម្អិត
- Summary Reports: មើលរបាយការណ៍សង្ខេប
- Users: គ្រប់គ្រងអ្នកប្រើ

## គោលការណ៍ប្រើប្រាស់

- បញ្ចូល master data ឲ្យត្រឹមត្រូវជាមុន
- កត់ត្រាប្រតិបត្តិការភ្លាមៗ កុំទុកចាំក្រោយ
- ពិនិត្យស្តុកមុនធ្វើ distribution ឬ dispensing
- ប្រើ batch និង expiry date នៅពេលទិន្នន័យទាំងនេះមានសារៈសំខាន់
- ពិនិត្យរបាយការណ៍មុន submit ឬ print

## លំដាប់ការងារដែលណែនាំ

1. បង្កើត Categories
2. បង្កើត Medicines
3. កត់ត្រា Distributions ឬ stock movement
4. កត់ត្រា Dispensing
5. កែសម្រួល Stock ប្រសិនបើមានភាពខុសគ្នា
6. បង្កើត Reports និង Summary Reports

## ចំណាំសម្រាប់អ្នកប្រើថ្មី

- សូមរៀន Medicines និង Stock ជាមុនសិន
- បន្ទាប់មករៀន Distributions និង Dispensing
- ចុងក្រោយរៀន Reports និង Users
