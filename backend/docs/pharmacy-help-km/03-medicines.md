# Help សម្រាប់ Medicines

## គោលបំណង

Medicines ប្រើសម្រាប់បង្កើត និងគ្រប់គ្រងព័ត៌មានស្តង់ដាររបស់ឱសថ។ ទិន្នន័យក្នុង module នេះគឺជាមូលដ្ឋានសម្រាប់ Stock, Distribution, Dispensing និង Reports។

## Fields សំខាន់ៗ

- Code: លេខកូដឱសថ
- Name: ឈ្មោះឱសថ
- Name (KH): ឈ្មោះជាភាសាខ្មែរ
- Category: ប្រភេទឱសថ
- Dosage form: ទម្រង់ឱសថ
- Strength: កម្រិតឱសថ
- Unit: ឯកតា
- Manufacturer: ក្រុមហ៊ុនផលិត
- Unit price: តម្លៃក្នុងមួយឯកតា

## របៀបបង្កើតឱសថថ្មី

1. ចូលទៅកាន់ Medicines
2. ចុចបង្កើតឱសថថ្មី
3. បញ្ចូល Code, Name និង Unit ជាចម្បង
4. បន្ថែម Category, Dosage form, Strength និងព័ត៌មានផ្សេងទៀត
5. ចុច Save

## ការណែនាំសម្រាប់ការបញ្ចូលទិន្នន័យ

- Code គួរតែមានទម្រង់ស្តង់ដារ និងមិនស្ទួន
- Unit គួរត្រូវនឹងការចែកចាយជាក់ស្តែង ដូចជា tablet, bottle, box
- Name (KH) គួរបញ្ចូលនៅពេលអ្នកប្រើភាគច្រើនប្រើភាសាខ្មែរ
- Unit price អាចជួយសម្រាប់ tracking និង report តម្លៃ

## ការកែប្រែ និងលុប

- អាចកែព័ត៌មានឱសថបាន ប្រសិនបើមានកំហុស
- មុនលុបឱសថ សូមពិនិត្យថាវាត្រូវបានប្រើក្នុង transaction រួចហើយឬនៅ

## បញ្ហាញឹកញាប់

- មិនអាច Save បាន ព្រោះខ្វះ Code ឬ Name
- Category មិនមានក្នុងប្រព័ន្ធ
- Unit បញ្ចូលមិនស្របនឹងការប្រើប្រាស់ពិត
