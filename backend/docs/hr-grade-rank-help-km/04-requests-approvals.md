# បង្កើតសំណើ និងអនុម័ត

## A. បង្កើតសំណើ

1. ចូល Tab **បញ្ជីគ្រប់លក្ខខណ្ឌ**
2. ជ្រើសមន្ត្រីដែលត្រូវស្នើ
3. ចូល Tab **បង្កើតសំណើ**
4. បំពេញព័ត៌មានស្នើសុំ៖
   - ប្រភេទសំណើ
   - ថ្ងៃមានប្រសិទ្ធភាព
   - លេខលិខិត/ថ្ងៃលិខិត (បើមាន)
   - កំណត់សម្គាល់
5. Submit

## B. ការអនុម័ត

1. ចូល Tab **ការអនុម័ត**
2. Filter តាមសិទ្ធិ និងស្ថានភាព
3. ពិនិត្យ target pay level មុនសម្រេច
4. ជ្រើសសកម្មភាព៖
   - អនុម័ត
   - បដិសេធ
   - ផ្តល់យោបល់បន្ថែម
5. កំណត់មូលហេតុបើបដិសេធ

## C. បន្ទាប់ពីអនុម័ត

- កំណត់ត្រានឹងចូល Tab **ប្រវត្តិដំឡើង**
- សំណើនឹងប្តូរស្ថានភាពទៅ approved
- របាយការណ៍នឹងរាប់តាមទិន្នន័យថ្មី
