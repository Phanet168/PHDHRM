# បញ្ហាញឹកញាប់ (FAQ)

## 1) មន្ត្រីមិនចូលបញ្ជីគ្រប់លក្ខខណ្ឌ

ពិនិត្យ៖

- ឆ្នាំ និង cutoff date
- ស្ថានភាពការងារ (active/inactive)
- ថ្ងៃដំឡើងចុងក្រោយ ឬថ្ងៃចូលបម្រើការងារ
- ប្រភេទបុគ្គលិក

## 2) សំណើស្ទួន

- កុំ submit ម្តងច្រើនដោយគ្មាន refresh
- ពិនិត្យ Tab សំណើទាំងអស់ មុនបង្កើតថ្មី
- ប្រើ action លុប/បោះបង់ សម្រាប់ draft ដែលមិនប្រើ

## 3) អនុម័តមិនបាន

- ពិនិត្យសិទ្ធិអ្នកប្រើ (`update_employee` + workflow role)
- ពិនិត្យថាសំណើនៅ stage ដែលអ្នកអាចសម្រេចបាន

## 4) Report ទិន្នន័យមិនត្រូវ

- ពិនិត្យ filter ឆ្នាំ
- ពិនិត្យថាសំណើបានអនុម័តរួច ឬនៅ pending
- ពិនិត្យថ្ងៃមានប្រសិទ្ធភាពក្នុងសំណើ

## 5) អក្សរខ្មែរ display ខូច

- ពិនិត្យ encoding UTF-8
- ពិនិត្យ font Khmer នៅ browser/server
- ពិនិត្យទិន្នន័យដើមក្នុង DB មិន mixed encoding
