# ន័យនៃ Tab នីមួយៗ

## 1) ផ្ទាំងសង្ខេប

- បង្ហាញ KPI សំខាន់ៗ (មន្ត្រីសរុប, គ្រប់លក្ខខណ្ឌ, សំណើកំពុងរង់ចាំ, លើស ៣ ឆ្នាំ...)
- មើលស្ថានភាពរហ័ស មុនចូលធ្វើការងារ

## 2) បញ្ជីគ្រប់លក្ខខណ្ឌ

- តារាងបុគ្គលិកដែលអាចដាក់សំណើបាន
- ជាចំណុចចាប់ផ្តើមសម្រាប់បង្កើតសំណើ

## 3) ព័ត៌មានមន្ត្រី

- មើលទិន្នន័យបច្ចុប្បន្ន រួមទាំងថ្នាក់បច្ចុប្បន្ន និងថ្ងៃដំឡើងចុងក្រោយ
- ប្រើពិនិត្យមុនធ្វើសំណើ

## 4) បង្កើតសំណើ

- បង្កើតសំណើថ្មី
- កែ target pay level និងឯកសារយោងមុនបញ្ជូន

## 5) ប្រវត្តិដំឡើង

- បង្ហាញការដំឡើងដែលបានអនុម័តរួច
- តាមដានប្រវត្តិមន្ត្រីម្នាក់ៗ

## 6) សំណើទាំងអស់

- មើលសំណើគ្រប់ស្ថានភាព (pending/approved/rejected...)
- ប្រើសម្រាប់តាមដានដំណើរការ

## 7) ការអនុម័ត

- ផ្ទាំងធ្វើសេចក្តីសម្រេច
- មាន action ជាក្រុម/ម្នាក់ៗ តាមសិទ្ធិ

## 8) របាយការណ៍

- បង្ហាញលទ្ធផលសម្រាប់ Export CSV/PDF និងបោះពុម្ព

## 9) ការជូនដំណឹង

- បង្ហាញករណីត្រូវតាមដាន (មិនទាន់ដំឡើង, គ្រោងដំឡើងជិតមកដល់...)
