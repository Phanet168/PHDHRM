# ៨) និវត្តន៍ និងអសកម្ម

## 8.1 គោលបំណង

ផ្នែកនេះសម្រាប់តាមដានមន្រ្តីជិតចូលនិវត្តន៍, ព្យាករណ៍ឆ្នាំខាងមុខ, និងគ្រប់គ្រងករណីអសកម្មឱ្យបានត្រឹមត្រូវ។

## 8.2 និវត្តន៍ (Retirement)

### អ្វីដែលត្រូវធ្វើ

1. ជ្រើសឆ្នាំគោលដៅ
2. ពិនិត្យបញ្ជីមន្រ្តីជិតនិវត្តន៍
3. ពិនិត្យទិន្នន័យ DOB និងថ្ងៃចូលបម្រើការងារ
4. បង្កើត/នាំចេញរបាយការណ៍

### របាយការណ៍ដែលគួរមាន

- បញ្ជីចូលនិវត្តន៍តាមឆ្នាំ
- ព្យាករណ៍ 1-5 ឆ្នាំ
- បញ្ជីចូលនិវត្តន៍រួច

## 8.3 ស្ថានភាពអសកម្ម (Inactive)

ករណីទូទៅ៖

- ផ្លាស់ចេញ
- ស្លាប់
- ឈប់បម្រើការងារ
- ព្យួរការងារ
- ទំនេរគ្មានបៀវត្ស

## 8.4 គោលការណ៍សំខាន់

- ការប្តូរអសកម្មត្រូវមានមូលហេតុច្បាស់
- ត្រូវកត់ត្រាក្នុងប្រវត្តិការងារ
- ត្រូវមានលិខិតយោង (បើ policy តម្រូវ)

## 8.5 Checklist

1. DOB ត្រឹមត្រូវ
2. ឆ្នាំគោលដៅត្រឹមត្រូវ
3. ស្ថានភាពអសកម្មមានមូលហេតុ
4. ប្រវត្តិការងារកត់ត្រារួច
