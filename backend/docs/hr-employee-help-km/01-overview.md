# бҹЎ) бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

## 1.1 бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҳбҹүбһјбһҢбһ»бһӣ

бһҳбҹүбһјбһҢбһ»бһӣ **бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ** бһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢ 4 бһ…бҹҶбһҺбһ»бһ…бҹ–

1. бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ“бҹ’бһҡбҹ’бһҸбһёбһұбҹ’бһҷбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ
2. бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбһ¶бһҳбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ (Traceable History)
3. бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡ бһҠбһјбһ…бһҮбһ¶ бһЎбһҫбһ„бһҸбһҪбһ“бһ¶бһ‘бһё бһЎбһҫбһ„бһҗбҹ’бһ“бһ¶бһҖбҹӢ бһ•бҹ’бһӣбһ¶бһҹбҹӢбһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ
4. бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸ

## 1.2 бһңбһ·бһҹбһ¶бһӣбһ—бһ¶бһ–бһҖбһ¶бһҡбһ„бһ¶бһҡ (Scope)

- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“
- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ
- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ·бһҖбҹ’бһҹбһ¶/бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ
- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҖбһ¶бһҡбһ„бһ¶бһҡ
- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҗбҹ’бһ“бһ¶бһҖбҹӢ бһ“бһ·бһ„бһӢбһ¶бһ“бһ“бҹ’бһҸбһҡбһҹбҹҗбһҖбҹ’бһҸбһ·
- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ“бһ·бһңбһҸбҹ’бһҸбһ“бҹҚ бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһўбһҹбһҖбһҳбҹ’бһҳ
- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҜбһҖбһҹбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢ бһ“бһ·бһ„бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ

## 1.3 бһӣбҹҶбһ бһјбһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ

3. бһ”бһҫбһҖ `Profile` бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“
4. бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҸбһ¶бһҳбһҖбҹ’бһҡбһ»бһҳбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’
5. бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢ бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
6. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹ… `бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ`
