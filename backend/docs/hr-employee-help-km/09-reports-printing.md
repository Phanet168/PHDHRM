# ៩) របាយការណ៍ និងបោះពុម្ព

## 9.1 គោលបំណង

សម្រាប់បង្កើតរបាយការណ៍ផ្លូវការរដ្ឋបាល និងបោះពុម្ពឯកសារអោយមានទ្រង់ទ្រាយត្រឹមត្រូវ។

## 9.2 ប្រភេទរបាយការណ៍សំខាន់

- បញ្ជីបុគ្គលិកតាមអង្គភាព
- របាយការណ៍ថ្នាក់/ឋានន្តរស័ក្តិ
- របាយការណ៍ឡើងតួនាទី
- របាយការណ៍និវត្តន៍

## 9.3 ជ្រើស Export ណាដែលសម

- **CSV/Excel**: សម្រាប់វិភាគ និងកែសម្រួលបន្ត
- **PDF**: សម្រាប់ឯកសារផ្លូវការ និងបោះពុម្ព

## 9.4 មុនបោះពុម្ព (Print Checklist)

1. ពិនិត្យ filter (ឆ្នាំ, អង្គភាព, ស្ថានភាព)
2. ពិនិត្យ Header/Title ត្រឹមត្រូវ
3. ពិនិត្យឈ្មោះ និងលេខអត្តលេខ
4. ពិនិត្យទំហំក្រដាស និង margin
5. Print Preview ម្តងមុនបោះពុម្ពពិត

## 9.5 ស្តង់ដារ Header/Footer (ណែនាំ)

- ចំណងជើងស្ថាប័ន
- ឈ្មោះរបាយការណ៍
- កាលបរិច្ឆេទបង្កើត
- អ្នករៀបចំ/អ្នកពិនិត្យ/អ្នកអនុម័ត

## 9.6 បញ្ហាដែលជួបញឹកញាប់

- អក្សរខ្មែរ​ខូចក្នុង PDF/Excel -> ពិនិត្យ font និង encoding
- Print layout រំខាន -> ពិនិត្យ scale (100%) និង paper size
- ទិន្នន័យខ្វះ -> ពិនិត្យ filter និងសិទ្ធិមើលទិន្នន័យ
