# бҹ§) бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ

## 7.1 бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„

бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһҳбһ“бҹ’бһҡбҹ’бһҸбһё бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ„бһ¶бһҡбһұбҹ’бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҸбһ¶бһҳбһӣбһ·бһҒбһ·бһҸбҹ”

## 7.2 бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ

- бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ–бһёбһўбһ„бҹ’бһӮбһ—бһ¶бһ– A бһ‘бҹ… B
- бһ•бҹ’бһӣбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһҹбҹӮбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбһјбһ…бһӮбҹ’бһ“бһ¶
- бһ•бҹ’бһӣбһ¶бһҹбҹӢбһҸбһ¶бһҳбһӣбһ·бһҒбһ·бһҸбһ”бһүбҹ’бһҮбһ¶бһҖбһ¶бһҡ

## 7.3 бһҮбҹҶбһ бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ

1. бһҮбҹ’бһҡбһҫбһҹбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ
2. бһҮбҹ’бһҡбһҫбһҹбһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһӮбҹ„бһӣбһҠбҹ… (Tree Combo)
3. бһ”бһүбҹ’бһ…бһјбһӣбһҗбҹ’бһ„бҹғбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
4. бһ”бһүбҹ’бһ…бһјбһӣбһӣбҹҒбһҒбһӣбһ·бһҒбһ·бһҸ бһ“бһ·бһ„бһҗбҹ’бһ„бҹғбһӣбһ·бһҒбһ·бһҸ
5. бһ”бһүбҹ’бһ…бһјбһӣбһҳбһјбһӣбһ бҹҒбһҸбһ»
6. Save

## 7.4 бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ•бҹ’бһӣбһ¶бһҹбҹӢ

- бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“ update
- бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳ transfer event
- бһҹбһ·бһ‘бҹ’бһ’бһ·бһ…бһјбһӣбһҳбһҫбһӣбһҸбһ¶бһҳбһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ (бһ”бһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбҹҶбһҺбһҸбҹӢ)

## 7.5 бһҖбҹҶбһ бһ»бһҹбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢ

- бһҮбҹ’бһҡбһҫбһҹбһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһӮбҹ„бһӣбһҠбҹ…бһҒбһ»бһҹ level
- бһӮбҹ’бһҳбһ¶бһ“бһҗбҹ’бһ„бҹғбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- Save бһҡбһҪбһ…бһҸбҹӮбһҳбһ·бһ“бһғбһҫбһүбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· (бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ conflict бһҗбҹ’бһ„бҹғ)

## 7.6 Checklist

1. бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһӮбҹ„бһӣбһҠбҹ…бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
2. бһҗбҹ’бһ„бҹғбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
3. бһҳбһ¶бһ“бһӣбһ·бһҒбһ·бһҸбһҷбҹ„бһ„
4. бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ„бһ¶бһҡбһҖбһҫбһҸ record бһҗбҹ’бһҳбһё
