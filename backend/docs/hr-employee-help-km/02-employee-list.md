# ២) បញ្ជីបុគ្គលិក

## 2.1 គោលបំណង

ផ្ទាំងនេះសម្រាប់ស្វែងរក និងគ្រប់គ្រងបុគ្គលិកជាប្រចាំថ្ងៃ។

## 2.2 ទិន្នន័យសំខាន់ក្នុងតារាង

- ល.រ
- អត្តលេខមន្រ្តី (10 ខ្ទង់)
- ឈ្មោះមន្រ្តី
- ភេទ
- ថ្ងៃខែឆ្នាំកំណើត
- តួនាទីបច្ចុប្បន្ន
- កាំប្រាក់/ថ្នាក់បច្ចុប្បន្ន
- ស្ថានភាពការងារ
- អង្គភាព
- សកម្មភាព (មើល/កែ/លុប តាមសិទ្ធិ)

## 2.3 របៀបស្វែងរកឱ្យលឿន

1. ស្វែងរកដោយអត្តលេខមុនគេ (ត្រឹមត្រូវខ្ពស់)
2. បន្ទាប់មកស្វែងរកដោយឈ្មោះខ្មែរ/ឡាតាំង
3. ប្រើ Filter អង្គភាព ដើម្បីកាត់ចំនួនលទ្ធផល
4. ប្រើ Filter ស្ថានភាព (សកម្ម/អសកម្ម)

## 2.4 តម្រងដែលគួរប្រើជាញឹកញាប់

- អង្គភាព
- ភេទ
- ប្រភេទបុគ្គលិក
- ស្ថានភាពការងារ
- កាំប្រាក់/ថ្នាក់

## 2.5 សកម្មភាពក្នុងជួរដេកម្នាក់

- **មើល (View)**: មើលព័ត៌មានលម្អិត និងប្រវត្តិ
- **កែប្រែ (Edit)**: កែព័ត៌មានតាមសិទ្ធិ
- **លុប (Delete)**: អនុញ្ញាតតែអ្នកសិទ្ធិខ្ពស់ និងត្រូវប្រុងប្រយ័ត្ន

## 2.6 ករណីប្រើប្រាស់ជាក់ស្តែង

### ករណី A: HR ត្រូវរកមន្រ្តីតាមលេខ

1. វាយអត្តលេខ 10 ខ្ទង់
2. ចុច View
3. ពិនិត្យក្រុមព័ត៌មានដែលត្រូវការ

### ករណី B: ត្រូវចេញបញ្ជីតាមអង្គភាព

1. ជ្រើសអង្គភាពក្នុង Filter
2. ចុចស្វែងរក
3. Export CSV/Excel តាមតម្រូវការ

## 2.7 បញ្ហាដែលជួបញឹកញាប់

- មិនឃើញទិន្នន័យ: ពិនិត្យ filter ដាក់ខុស
- លទ្ធផលតិចពេក: Clear filter រួចស្វែងរកម្តងទៀត
- អក្សរខ្មែរ​ខូច: ពិនិត្យ encoding/locale/font
